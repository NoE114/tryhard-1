## Project Template

When starting a new project, include in README.md:

1. **Description**: What does this project do?
2. **Skills Applied**: Which Python concepts are used?
3. **Installation**: How to set up the project?
4. **Usage**: How to run the project?
5. **Challenges**: What was difficult?
6. **Learnings**: What did you learn?
7. **Future Improvements**: What could be enhanced?
